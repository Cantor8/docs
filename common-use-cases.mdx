## **What it is**

C8 WalletConnect SDK is a ready-made integration layer that allows third-party applications to connect their users to C8 Wallet on the Canton network.

It gives partner applications a secure and familiar way to work with wallet-based user approvals, Canton assets, and transaction flows without building wallet infrastructure from scratch.

## **Why it matters**

For most teams, building a wallet layer is expensive and operationally sensitive. It requires authentication flows, user approval screens, transaction signing, wallet state handling, and network-specific logic.

C8 WalletConnect SDK removes that complexity. Partners can focus on their product experience while C8 Wallet handles the wallet interaction layer.

## **Who it is for**

The SDK is designed for development teams building applications that need Canton wallet connectivity.

It is especially relevant for:

- DeFi applications
- Payment and settlement products
- Treasury and asset management tools
- Trading or investment interfaces
- Institutional finance workflows
- Applications that need user-approved asset movement on Canton

## **How partners can use it**

Partners can embed C8 Wallet connectivity into their own application experience. A user can connect their wallet, review available assets, approve actions, and follow transaction progress while staying inside the partner product journey.

The SDK is useful when an application needs to support wallet-based actions but does not want to own private key management, transaction signing, or direct Canton ledger communication.

## **Business value**

C8 WalletConnect SDK helps partners launch Canton-enabled products faster, reduce integration risk, and provide users with a trusted wallet experience backed by Cantor8 infrastructure.

In short: it turns C8 Wallet into a plug-in wallet layer for partner applications.
