Cantor8 Enterprise Wallet (`wallet-api`) is a **Scala-based service** that provides a high-level HTTP API for wallet operations on the Canton Network. It wraps the [Wallet SDK](https://github.com/Cantor8/wallet-sdk/) to communicate securely with the [Wallet Backend](https://github.com/Cantor8/wallet-backend/) (Canton Ledger).

It is built for institutions that need a hardened, OIDC-secured, high-throughput wallet service backing their treasury, settlement, or custody workflows — without owning the underlying ledger plumbing.
